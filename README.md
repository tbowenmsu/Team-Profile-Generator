# Team-Profile-Generator
![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)
  ## Description
   A Node.js command-line application that takes in information about employees on a software engineering team, then generates an HTML webpage that displays summaries for each person. You will find in this repo tests as well to ensure that it passes and works as intended.
